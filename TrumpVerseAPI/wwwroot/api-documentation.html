TrumpVerse API Documentation

Detailed overview of how to use the endpoints in the TrumpVerse API

API Base URL

The API is available at: http://localhost:5191/api

Endpoints

Get All Staff

Method: GET

Endpoint: /TrumpStaff

What it does: Retrieves a list of all staff members in the database.

Get Staff by ID

Method: GET

Endpoint: /TrumpStaff/{id}

What it does: Fetches details for a specific staff member using their unique ID.

Create New Staff

Method: POST

Endpoint: /TrumpStaff

What it does: Adds a new staff member to the database. This includes choosing a profile picture from a pre-defined set.

Update Staff

Method: PUT

Endpoint: /TrumpStaff/{id}

What it does: Updates information about a staff member. Useful for changes to roles or updating contact info.

Delete Staff

Method: DELETE

Endpoint: /TrumpStaff/{id}

What it does: Removes a staff member from the database. Be carefulâ€”this action is permanent!

Get Available Images

Method: GET

Endpoint: /FileUpload/available-images

What it does: Returns a list of profile pictures available for staff members. These images are pre-stored on the server.
